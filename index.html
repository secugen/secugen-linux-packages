SecuGen Linux Packages

Public APT and RPM repositories for SecuGen Linux package distribution.

Install On Debian Or Ubuntu

First-time setup 
curl -LO https://secugen.github.io/secugen-linux-packages/secugen-linux-repo.deb
sudo apt install ./secugen-linux-repo.deb
sudo apt update
sudo apt install secugen-connect
Update after setup 
sudo apt update
sudo apt install secugen-connect
Prerequisites
  • Use Debian or Ubuntu on amd64.
  • curl is needed to download the setup package.
  • ca-certificates is installed by the setup package if needed.
  • libusb-1.0-0 is installed automatically with secugen-connect.
  • The package installs udev rules and the system service. Reconnect the reader or reboot if a USB reader was already attached during installation.

Install On Fedora

First-time setup 
sudo curl -fsSL -o /etc/yum.repos.d/secugen.repo https://secugen.github.io/secugen-linux-packages/secugen.repo
sudo dnf makecache
sudo dnf install secugen-connect
Update after setup 
sudo dnf makecache
sudo dnf upgrade secugen-connect
Prerequisites
  • Use Fedora on x86_64.
  • curl is needed to install the repo file.
  • libusb1 is installed automatically with secugen-connect.
  • The package installs udev rules and the system service. Reconnect the reader or reboot if a USB reader was already attached during installation.
  • No separate Linux device-driver package is required.
